**KingsCrowd Take Home**

Scry is a dashboard for viewing all active raises on the KingsCrowd platform! You can run it locally by taking the
following steps:

1. Git Clone the repository
2. CD into the scry directory
3. Run an NPM Install using the command npm install
4. Run the project using the command npm run dev
5. Navigate to the url given in the terminal, http://localhost:3000/ by default

****External Dependencies:****

* Nuxt
* Tailwind

******Testing******

* Playwright
* a11y
